## WeatherShpere Web Application
## How to Install and Run

1-Clone the repository
Open up command prompt in your laptop and type this and press enter:

git clone https://github.com/Safa448/WeatherSphere-Web-Application.git

2-Go to folder
In the command prompt (after pasting the github link) type the below information:

cd WeatherSphere-Web-Application

3-Install requirements
In the command prompt(after doing the first 2 steps) type this and press enter:

pip install -r requirements.txt

This will install all the requirements needed to run the web app.

4-Run app
In the command prompt, type this to run the application and press enter:

python app.py

5- Open browser
Open your preferred web browser and paste the below link and press enter:
http://127.0.0.1:5000
